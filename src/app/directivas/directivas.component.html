06, 07 - Directivas: estructurales, de atributos y más

Directivas estructurales:

Se refiere a que pueden modificar elementos del DOM, por ejemplo, ocultar o mostrar una etiqueta HTML, un ejemplo de directivas estructurales son: - ngFor - ngIf

ngFor:

En este caso para poner un ejemplo de ngFor, tenemos un array declarado y definido en nuestra clase y lo vamos a renderizar con la directiva ngFor.

  • {{ city }}

ngIf:

Ahora vamos a poner en practica el ngIf, tenemos un input el cual tiene una directiva ngModel, asociada a una propiedad llamada name, y tenemos una etiqueta 'p' la cual se va a mostrar solo cuando la propidad input tenga algun valor

Tu nombre es: {{ name }}

Directivas de atributos

Para añadir una directiva de atributos a un elemento se debe utilizar la sintaxis de corchete

ngClass

Con esta directiva podemos colocarle clases a los elementos

  • {{ student }} (clases colocadas con un string) [ngClass]="'selected color-white'"
  • {{ tutor }} (clases colocadas con un array) [ngClass]="['selected', 'color-white']"
  • {{ pay }} (clases colocadas con un objeto) [ngClass]='selected': false}

ngStyle

  • {{ pay }}
